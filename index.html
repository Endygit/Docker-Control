Fonctionnalités

Tout ce dont vous avez besoin
dans une seule interface

De la gestion des conteneurs au terminal intégré, Docker Control centralise tout votre workflow Docker.

📦

Gestion des conteneurs

Démarrez, arrêtez, redémarrez et créez vos conteneurs. Stats CPU/RAM en temps réel pour chaque instance.

🏷️

Images Docker

Parcourez vos images locales, pullez depuis Docker Hub avec une barre de progression, inspectez les layers.

💾

Volumes & Données

Créez, montez et gérez vos volumes. Visualisez les mountpoints et les métadonnées de chaque volume.

🌐

Réseaux

Inspectez les subnets, les conteneurs connectés et les configurations IPAM. Créez des réseaux custom.

🖥️

Shell intégré multi-onglets

Vrai terminal PTY avec xterm.js. Plusieurs onglets simultanés, 5 thèmes visuels (Amber Rétro, Matrix, Océan, Dracula, Solarized), commandes rapides intégrées : ping, ssh, curl, nmap, docker stats…

📋

Logs en temps réel

Streaming des logs avec auto-scroll, distinction stdout/stderr et timestamps.

⚡

Animation de démarrage

Boot sequence cinématique avec particules, cubes animés et terminal de chargement.

🔌

Connexion locale & distante

Socket détecté automatiquement (Docker Desktop, Colima, OrbStack). TCP distant avec TLS optionnel.

Tout ce que vous devez savoir avant de commencer.

Oui, Docker Control est entièrement gratuit et open source (MIT). Utilisable pour des projets personnels et professionnels, sans restriction.
Supporte Windows 10/11 (x64, ia32), macOS 10.15+ (Intel x64 et Apple Silicon M1/M2/M3), et Linux (x64 et arm64) via AppImage, .deb et .rpm.
Oui, Docker Control est une interface graphique qui se connecte à un daemon Docker existant. Installez Docker Desktop ou Docker Engine avant de lancer l'application.
Dans l'écran de connexion, choisissez l'onglet Distant (TCP). Entrez l'IP et le port (2375 sans TLS, 2376 avec TLS). Activez TLS et renseignez les certificats pour sécuriser la connexion.
Oui ! Le shell utilise un vrai PTY (node-pty + xterm.js). Il supporte vim, nano, htop, ssh, tmux et toute application interactive.
Dans la vue Shell, cliquez sur le bouton Thème dans la barre d'actions. Cinq thèmes disponibles : Amber Rétro, Matrix Vert, Océan, Dracula, Solarized. Le changement s'applique à tous les onglets instantanément.
Absolument. Téléchargez les sources, installez Node.js 18+ et exécutez npm install puis npm start. Pour builder : npm run build:linux, npm run build:mac ou npm run build:win.
Non. L'application est 100% locale. Elle ne collecte aucune donnée et ne contacte aucun serveur externe. Aucune connexion internet n'est requise pour fonctionner.
